source_csv - README

Chứa các file CSV dùng để huấn luyện và thử nghiệm.

Yêu cầu format:

- Columns: soil_moisture, temperature, humidity, label
- Nếu file có header sai (`lable`), sửa hoặc chạy script tiền xử lý.

Gợi ý preprocessing:

- Loại bỏ giá trị thiếu, outlier, và scaling sau khi chia train/test.
